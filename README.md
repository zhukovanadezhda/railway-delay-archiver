# How to run the project

## 1. Get an SNCF / Navitia API token
1. Create an account at: https://www.navitia.io
2. Generate an API token
3. Create a `.env` file at the project root containing:

```bash
SNCF_TOKEN=your_api_token_here
```

## 2. Create a Conda environment from the provided file and run main.sh
